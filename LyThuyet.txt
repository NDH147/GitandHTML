Để chạy được git trong webstorm cần thực hiện 2 câu lệnh:
git config --global user.email <email in github>
git config --global user.name <username in github>

local repository: nơi lưu trữ mã nguồn trên máy tính
remote repository: nơi luw trữ mà nguồn online

git innit: khởi tạo local repository
git add: đưa các tháy đổi vào staging area
git comit: đưa thay đổi từ staging vào local repo, lưu lại lịch sử tháy đổi
git remote: kết nối với remote repo
git push đưa các thay đổi từ local lên remote
